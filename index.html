  • Suduko Solver: Speed and Simplicity

    In this personal project, I developed an efficient Sudoku solver that stands out for its speed and simplicity. Inspired by Tom Dillon's Tduko, which uses advanced techniques and bitwise operations in C for optimal performance, I aimed to create a high-speed solver in Python using more straightforward methods. On even the most difficult puzzles, the solver handles puzzles in approx. 0.05 seconds!

  • Solving the Rubik's Cube with Maths

    I approach complexity of the Rubik’s Cube by leveraging H. Kociemba's Two-Phase algorithm to navigate the vast solution space efficiently with my pure-Python implementation. Given the cube’s 43 quintillion possible states, finding the singular solution might seem daunting, but the Two-Phase algorithm uses advanced Group Theory to collapse the complexity and navigate the sea of scrambles.
